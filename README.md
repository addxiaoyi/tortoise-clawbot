# 乌龟助手

一款功能强大的 AI 助手桌面应用。

## 下载

请前往 [Releases](https://github.com/addxiaoyi/tortoise-clawbot/releases) 页面下载最新版本。

## 支持的平台

- Windows x64
- Windows x86
- Linux x64
- Linux x86
- Linux ARM64
- macOS x64
- macOS ARM64

## 系统要求

### Windows
- Windows 10 (版本 1803+) 或更高版本
- WebView2 运行时

### Linux
- GTK 3.24+
- WebKitGTK 4.1+
- GLib 2.56+

### macOS
- macOS 10.15 (Catalina) 或更高版本
