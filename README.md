# 乌龟助手 (Tortoise)

一款功能强大的 AI 助手桌面应用。

## 下载

请前往 [Releases](https://github.com/addxiaoyi/tortoise-clawbot/releases) 页面下载最新版本。

## 支持的平台

- Windows x64 / x86
- Linux x64 / x86 / ARM64
- macOS x64 / ARM64

## 系统要求

### Windows
- Windows 10+ with WebView2 runtime

### Linux
- GTK 3.24+, WebKitGTK 4.1+

### macOS
- macOS 10.15+
